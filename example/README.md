### Demo

A live demo is available [here](https://express-ugc-proxy-demo.xboxreplay.now.sh).

### Installation

```bash
$ npm install
```

### Start

```bash
$ XBL_EMAIL=xbl-account@domain.com \
  XBL_PASSWORD=********* \
  npm start
```

### Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download))

```bash
now
```
